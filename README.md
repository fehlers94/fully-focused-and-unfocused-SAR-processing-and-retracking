# fully-focused-and-unfocused-SAR-processing-and-retracking

## About
This repository contains the MATLAB code and software scripts used to generate the data in the work of Ehlers et al. (2023) *Validity assessment of SAMOSA retracking for fully-focused SAR altimeter waveforms* (https://doi.org/10.1016/j.asr.2022.11.034) and Ehlers et al. (2023) *A noise autocovariance model for SAR altimeter measurements with implications for optimal sampling*, see references below and details in the source publication.
